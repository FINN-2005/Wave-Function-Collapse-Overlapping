# Wave-Function-Collapse-Overlapping

Implementation of a Overlapping version of Wave Function Collapse in pygame

## Installation

```bash
git clone https://github.com/FINN-2005/Wave-Function-Collapse-Overlapping.git
```
(optional)
```bash
pip install git+https://github.com/FINN-2005/pygame_template.git
```


## Usage

```python
python main.py
```
